# Nightly-Custom-Cursor-v2


    Кастомный курсор в чёрном цвете, который был сделан специально для Nightly-Custom-Menu

    Чтобы установить, вам придётся:
    1. Установить архив
    2. Открыть папку
    3. Нажать правой кнопкной мыши по файлу: Install.inf
    4. Всё подтвердить
    5. Поменять курсор
